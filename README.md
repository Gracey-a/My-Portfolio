My personal developer portfolio – modern, dark‑themed, responsive, and built to impress recruiters and clients.

**🔗 [Live site](https://aghogracious-portfolio.netlify.app)** · **📁 [GitHub Repo](https://github.com/Gracey-a/My-Portfolio)**

---

## ✨ Features

- **Hero section** with profile image and call‑to‑action
- **About me** – bio and current goals
- **Projects catalog** – 4 featured projects with live demo and GitHub links
- **Fun facts** – humanises the portfolio
- **Skills & expertise** – tech stack and learning background
- **Contact form** – integrated with Netlify Forms (email notifications set up)
- **Custom thank‑you page** – after form submission
- **Fully responsive** – works on mobile, tablet, desktop
- **Smooth animations** – fade‑up on scroll, hover effects

---

## 🛠️ Built with

- **HTML5**
- **Tailwind CSS** (via CDN)
- **JavaScript** (ES6)
- **Lucide icons**
- **Netlify** (hosting + form handling)
